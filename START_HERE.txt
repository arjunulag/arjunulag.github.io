METHOD 1 (Easiest - Windows):
  1. Double-click "START_SERVER.bat"
  2. Browser will show: http://localhost:8000
  3. Click "Sample SVGs" to try examples
  4. Or upload your own SVG file!

METHOD 2 (Manual Server - Recommended):
  1. Open Command Prompt or PowerShell in this folder
  2. Run: python -m http.server 8000
  3. Open browser and go to: http://localhost:8000
  4. Click "Sample SVGs" or upload your own!

METHOD 3 (Direct File - Limited):
  1. Double-click "index.html"
  2. Upload your own SVG files (works fine)
  3. Sample SVGs may not work (browser security)
  4. Use METHOD 1 or 2 for full features

METHOD 3 (With Node.js):
  1. Open Command Prompt or PowerShell in this folder
  2. Run: npm start
  3. Browser will open automatically
